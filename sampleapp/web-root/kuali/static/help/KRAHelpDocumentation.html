KRA Budget: PARAMETERS

 

The Budget Parameters page initiates the KRA budget. Values entered on this page determine calculations on other KRA budget pages, so required data elements on the Budget Parameters page must be completed before the user can work on any other page in the KRA budget.

 

General

 

The following data elements are required before the user can advance to any other page in the KRA Budget:

 

         Document Overview Description

         Project Director(s)

         Budget Type

         Personnel Inflation Rate (even if the rate is 0.00)

         Non-Personnel Inflation Rate (even if the rate is 0.00)

         Agency Name

         At least one Budget Period

         At least one Task

 

1.      Click the PD Lookup magnifying glass button to search for the Project Director's name when creating a new Budget. You may also change the Project Director's name on an existing Budget, if necessary, by choosing this option.

2.      Enter the inflation rate for Personnel expenses. Enter a number here, even if the number is 0.00.

3.      Enter or modify the Budget Name/Description (optional).

4.      Enter the inflation rate for Non-Personnel expenses. Enter a number here, even if the number is 0.00.

5.      Enter or modify the Program Announcement Name or Number (optional).

6.      Click the Agency Lookup magnifying glass button to search for an Agency name when creating a new Budget. You may also change the Agency name on an existing Budget, if necessary, by choosing this option.

7.      Click the Federal Pass Through Lookup magnifying glass if the Budget involves Federal Pass Through funds. This lookup allows you to search for, and select, the prime federal agency.

8.      Enter or modify the Grant Number (optional).

9.      Click the check box next to Institution Cost Share, if the budget will include cost sharing borne by your institution.

10. Click the check box next to 3rd-Party Cost Share, if the budget will include cost sharing borne by a third party.

11. Use the checkbox to select the Budget Type: Project, Subaward/Consortium, or Modular.

1.      Lookup a Project Director's Name by selecting the PD Lookup magnifying glass button. The Project Director Lookup screen will appear. You must enter at least one character in one of the fields to begin a search. More criteria will give you narrower results. Enter any combination of the following search criteria:

a.      Enter the Project Directors Kuali User ID. e.g., JOSMITH

b.      Enter the numeric University ID.

c.      Enter the numeric Empl ID.

d.      Enter the Project Director's name. e.g., JOHN SMITH

e.      Enter the alphanumeric Dept ID.

f.        Enter the Employee Status.

g.      Enter the Employee Type.

A wildcard search can be performed on any of the lookup fields.

2.      Select the Search button after entering the criteria. This will generate the results.

3.      Select a Project Director from the list of results and click the Return Value link. Once the Project Director is selected you will return to the previous screen.

4.      Select the Clear button to clear the fields of any previously entered criteria and begin a new search.

5.      Select the Cancel button to clear all unsaved information and return to the previous screen.

6.      If the Project Director will be named at a later time, select the Name Later button.

Budget Type

 

1.      Select the Budget Type by clicking the checkbox next to Project, Subaward/Consortium, or Modular.

a.      Project: the budget you are working on is the lead budget on the proposal

b.      Subaward/Consortium: the budget you are working on is a subaward or consortium budget on the proposal

c.      Modular: the sponsor is an NIH institute or center, and you are using the Modular Budget Format

 

The Modular Budget Type is only active when the agency is an NIH institute or center. When the Modular Budget Type is chosen, the Modular Budget Tab becomes available for data entry and Modular Budget creation.

 

Cost Share

 

1.      Click the Institution Cost Share check box if the budget includes Institution Cost Share (system defaults to Unchecked).

2.      Click the 3rd Party Cost Share check box if the budget includes third party Cost Share; this includes Subcontractor Cost Share (system defaults to Unchecked).

 

When these checkboxes are left at their default (unchecked), cost share budget entry lines are grayed-out on the Personnel and Non-Personnel budget screens.

 

Budget Periods (required)

 

At least one Budget Period must be added to the KRA budget. There can be no gaps or overlaps in dates between Budget Periods. A budget period must be at least one day, and no longer than 12 months.

 

1.      Enter a From Date (start date) and To Date (stop date) for at least one period in the designated format (mm/dd/yyyy) or by choosing the calendar option. If the calendar option is chosen, simply click on the desired date to fill in this field.

2.      Select the Add button to add the period to the budget.

3.      Perform actions 1 & 2 to add additional periods to the budget, up to 20 periods.

4.      Select the Delete button to delete the last added Period. You will be asked to confirm that youd like to delete the budget period.

5.      If your budget will include participants or trainees, enter the total number of participants or trainees per period on each budget period line. Do not enter a number for participants or trainees if your budget will not contain budgeted items for these categories.


Tasks / Components (required)

 

At least one Task, which can be renamed, is required. If your Budget requires more detail, you may add Tasks as necessary. A Task/Component is a portion of the Budget within a Budget Period that might require separate accountability. Some examples include having separate Projects or Cores, keeping track of projects with multiple indirect cost rates, or assigning different sections of the budget to multiple PDs.

 

1.      Select the Add button to add the initial required task, and to add any other tasks (up to 20).

2.      The default name can be overwritten; the field will hold up to 20 characters.

3.      If you wish to delete a given Task, select the Delete Task button next to the line for that Task.

4.   Click the checkbox for Occurs on Campus if the Task activity occurs On-Campus.  The default is checked.  Uncheck the box if the task occurs Off-Campus.

5.   If the KRA Budget is an NIH Modular Budget, the Modular Variance is automatically assigned to the first task.  If you want to assign the variance to another task, click the radio button to include the variance on the desired task.  The Modular Variance is the amount the KRA budget adds, as a placeholder, to the detailed budget to bring the total to the nearest modular amount. 

 

Fringe Benefit Rates / Research & Grad Assistant Fringe Benefit Rates

 

The KRA budget is automatically loaded with System Rates for Fringe Benefits and Graduate Assistant Fringe.  System Rates are the rates negotiated for your institution, and are maintained at a central level.  On proposals for sponsored awards, it may be necessary to change the Fringe Rates to accommodate regulations by a sponsor. 

 

1.      To change rates manually, enter the new rate in the Current Budget Rates field. (see step 3 below) 

2.      To change Current Rates back to the System Rates, click the Copy System Rates button .

3.      If a Current Rate has been manually modified, you must provide justification for this action. Enter this information in the Benefit Rate Change Justification text box.

4.      Select the Save button to save this information and return to the previous screen.

5.      Select the Reload button to clear this information and return to the previously saved state.

6.      Select the Close button to return to the previous screen. You will be prompted to save your changes or cancel them prior to returning to the previous screen.

 

Save 

 

Select the Save button to save your information. Any errors on the form will appear in red text next to the incorrect fields.  Clicking the Save button will take you to the Budget Overview page. 

 

KRA Budget: BUDGET OVERVIEW

 

The Budget Overview page allows you to view cumulative budget information for:

 

  • the entire project (all tasks and all periods)
  • any single task for any single period
  • any single task for the entire project period

 

The default view for the Budget Overview page shows cumulative budget information for Task 1, Period 1.


General

 

The Budget Overview page displays budget information as entered and modified on the Personnel, Non-Personnel, and Indirect Costs screens. Budget items can only be modified on the Personnel, Non-Personnel, and Indirect Cost screens, not from the Budget Overview page.

 

This page displays budget data both for amounts requested from the sponsoring agency, and for institution and/or third-party cost sharing. If the cost-sharing indicators for Institution Cost Sharing or 3rd-Party Cost Share are left unselected on the Budget Parameters page, the Budget Overview page will indicate that Cost Sharing is set to No.

 

Task / Period Selection

  

Use the drop down lists to make your Task Selection and/or Period Selection. Click the Update View button to change the screen display to your selections. The Update View button must be clicked; simply changing the drop-down menus will not change the screen view.

 

View Budget Detail

 

There are a couple of ways you can expand the Budget Overview page to view budget detail:

 

  1. Click the Expand All button to open all budget categories. This button, when clicked, will display all budget data entered on the Personnel and Non-Personnel screens, for all categories and sub-categories. Click the Collapse All button to close all the budget categories.
  2. Click the Show button next to the budget category you want to open. The Show button opens only the category you select. Any number of categories can be open or closed at one time. Click the Hide button on an opened category to close it.

Complete

 

The Budget Document will remain in the initiators action list, and cannot be attached to a Kuali Routing Form for routing/approvals until the Complete button is selected. When the budget is final, click the Complete button to initiate FYI notices to those on the Permissions list, and to enable you to link the Budget with a Routing Form. When you click the Complete button, a confirmation screen will appear. Click Yes if the budget is final. Click No if the budget is not yet complete. Once the budget is designated as Complete, its status cannot be changed. You would need to template the budget, and create a new one, in order to edit it.

KRA Budget: PERSONNEL

 

  1. The Project Director (added on the Budget Parameters page) is automatically added to the Personnel budget.  The Project Director can only be changed from the Budget Parameters page, not the Personnel screen. 
  2. Click the Lookup magnifying glass button to perform a lookup to add other personnel.  The Universal User Lookup screen will appear.  Perform the lookup to add additional personnel.

14. Click the Save button to save your edits.  The page will refresh with the following statement:  Document was successfully saved.

16. To delete an individual name from the project budget, click the checkbox next to the persons name, then select the Delete Selected  button.  Click the Save button to save the deletion.  Click the Reload button (before you Save) to restore the deleted person.

KRA Budget: NONPERSONNEL

 

  1. Choose the Task and Period you want to update by selecting them from the drop-down lists.  Click the Update View button to display the proper task and period.

 

  1. Select one of the Non-Personnel expense categories to edit by clicking the Show button next to the one you want to edit.  The Non-Personnel expense categories are:
    1. Consultants
    2. Equipment
    3. Supplies
    4. Travel
    5. Patient Care Costs
    6. Fellowships
    7. Participant Expenses
    8. Other Expenses
    9. Subcontractors

 

3.      Once a Non-Personnel expense category panel is opened, you can edit and add budget lines for the category.

Click the Return button to return to the previous screen.

11. To delete a line, click the Delete button next to that line.

12.   Click the Recalculate button to compute additions to the non-personnel budget.

13. Select the Save button to save your information.  The page will refresh with the following statement:  Document was successfully saved. 

14. Select the Reload button to clear all information and return to the previously saved state.

KRA Budget: NONPERSONNEL COPY OVER

  1. Add a nonpersonnel budget line as described in the Nonpersonnel budget instructions. 
  2. Once the budget line is added, click the Copy to Future Periods checkbox, then click View Period Allocation. From this screen, you can view the amount applied to future budget periods, with inflation added at the percentage indicated on the budget parameters page.  You cannot edit amounts from the View Period Allocation Screen.
  3. To delete a copied-forward expense from a future period, click the checkbox next to the expense in the View Period Allocation screen, then select Recalculate. This will remove the total amount from the selected period.  Click Return to go back to the Non-Personnel budget detail screen.
  4. Changes/deletion of the amount from budget period 1 can only take place in the Non-Personnel budget detail screen.  If the expense has already been copied forward, and you edit the amount, these changes will not copy over into future periods.  To extrapolate the changes to future periods, delete the expense line and add a new one, copying over the expense as indicated in step 2. 
  5. To change (but not delete) copied forward amounts for future periods, select the budget period from the drop-down menu on the Non-Personnel budget detail screen.  Click update view to access the detail screens and make changes for that period.  Changes will only affect the period you select; they will not be copied forward to later periods. 
  6. To remove the copied forward expense from all periods, access the first budget period Non-Personnel budget detail screen.  Click the Delete button next to the line you want to remove. 

 

KRA Budget: COST SHARE

 

The Cost Share screen allows you to indicate the distribution of Institution and Third Party Cost Share and shows a summary of the total Cost Share on your budget.

Cost Sharing amounts will display on the Cost Share page when:

 

         You indicate Yes  to budget includes Institution Cost Share on the Budget Parameters page, OR

         You indicate Yes to budget includes third party Cost Share on the Budget Parameters page, AND

         You enter Personnel and Non-Personnel cost sharing amounts in the budget screens

 

Cost Share amounts must be fully distributed.  Follow these steps to distribute cost sharing:


Cost Share - Institution Direct

 

1.      All cost share dollars must be matched to a source (distributed).  Cost sharing amounts are edited from the Personnel or Non-Personnel budget detail screens only. 

2.      All personnel Cost Share will be automatically distributed to the Chart/Org Code assigned on the individual's personnel detail screen. If there are several individuals with Cost Share in the same Organizational Code, their Cost Share amounts will be totaled and distributed on one line.

      The Chart/Org for Personnel is edited on the Personnel screens only.

3.      Non-personnel Cost Share amounts will be displayed on the "Balance To Be Distributed" line.

4.      The balance can be distributed across multiples sources. To distribute the balance, click the Add button in the Add Source section of the Cost Share page. The Organization Lookup screen will appear.

5.      Select a source Organization. Once an Organization is selected, you will return to the previous screen. Repeat steps 3 and 4 as many times as is necessary to include all source Organizations.

6.      Distribute the remaining dollars to the appropriate Chart of Accounts, Organizational Code, and Period. Enter the amounts in the Period field for each source line.

7.      Select the Recalculate button to update the amounts.

8.      Enter a brief description of the Cost Share in the Description field for each source line.

9.      To delete an added source line, click the Delete button next to the line you want to delete. The total will recalculate automatically.

 

Cost Share - Institution Indirect

 

Some agencies allow indirect costs to be charged against the direct cost share portion of a budget. These amounts are not editable and pull from the Indirect Cost on Institution Cost Share column on the Indirect Costs page.

 

Cost Share  3rd Party Direct

 

1.      All cost share dollars must be matched to a source (distributed).  3rd Party cost share amounts are edited from the Non-Personnel budget detail screens.

2.      The balance can be distributed across multiple sources. To distribute the balance, click the Add button in the Add Source section.

3.      Enter the name of the source in the Source field. Repeat steps 1 and 2 as many times as is necessary to include all 3rd party sources.

4.      Distribute the remaining dollars to the appropriate 3rd Party Source and Period. Enter the amounts in the Period field for each source line.

5.      Select the Recalculate button to update the amounts.

6.      Enter a brief description of the Cost Share in the Description field for each source line.

7.      To delete an added source line, click the Delete button next to the line you want to delete. The total will recalculate automatically.

 

Save / Reload

1. Select the Save button to save your information.

2. Select the Reload button to clear all editable fields and return to the previously saved state.

 

KRA Budget: MODULAR (NIH only)

 

The KRA Modular screen calculates and displays modular direct cost totals, for each budget period, based on the budget detail entered in the Personnel and Non-Personnel entry screens.  Modular amounts are calculated by averaging the total direct costs for all budget periods/tasks and rounding to the nearest modular amount (one module = $25,000).  The amount added/subtracted to/from each budget period (to arrive at the modular amount) is displayed in the Task selected on the Budget Parameters page.  This amount is only a placeholder.  The modular calculation method creates modular amounts that are level from budget period to budget period, although it is possible to change each periods modular amount individually.  The modular budget page includes text boxes to enter Personnel justification, Consortium justification, and Variable Adjustment justification. 

 

  1. Actual Direct Costs displays the total direct costs for each budget period, as calculated from Personnel and Non-Personnel budget entries.
  2. Modular Direct Cost is the amount calculated by averaging the total direct costs from all periods, and rounding to the nearest modular amount.
  3. If you need to change the Modular Direct Cost amount calculated by the KRA budget, click the Drop-Down menu to select another modular amount.  This may be necessary if costs will occur in some years but not others, like large equipment costs, etc.  Click the Recalculate button.
  4. Click the Show button, or Expand All button to open the Explanation entry panels. 
    1. Explanation-Personnel:  Enter or cut-and-paste the Personnel justification as required by NIH modular budget instructions.
    2. Explanation-Consortium:  Enter or cut-and-pate the Consortium justification as required by NIH modular budget instructions, if the budget includes Subcontracts.
    3. Explanation-Variable Adjustment:  Enter or cut-and-paste the justification required if module requests vary from year-to-year as required by NIH modular budget instructions. 
  5. Select the Save button to save your information
  6. Select the Reload button to clear all editable fields and return to the previously saved state.

 

 

KRA Budget: INDIRECT COST 

 

  1. Indicate the primary purpose of your project from the following:
    1. Research
    2. Instruction
    3. Other
  2. Indicate the current base indicator from the following:
    1. Standard Base (MTDC) calculates indirect costs automatically based on your institutional definition of MTDC, with exclusions
    2. Total Direct Cost (TDC) calculates indirect costs automatically on the direct costs total, with no exclusions
    3. Manual Base allows you to enter the indirect costs base when MTDC or TDC do not apply
  1. Indicate the indirect cost rate source from the following:
    1. Provided by the system the rate used will be from the institutional rate agreement
    2. Provided manually the user will supply the indirect cost rate
  1. Select the Manual Base option to manually enter the base indicator amount in the Manual Base ($) column of the grid.
  2. If the budget includes cost sharing:
    1. Click the Include Indirect Cost button if the final budget should display indirect costs on cost share totals.
    2. Click the Include Unrecovered Indirect Cost button if the final budget should display indirect costs foregone as a result of accepting a lower indirect cost rate. 
  1. Select the Recalculate button to redistribute the data based on the options selected.
  2. If the Manual Base indicator has been selected, enter the amount in the Manual Base ($) column of the grid under Agency Amount Requested.
  3. If the Provided Manually indicator has been selected, enter the Indirect Cost rate (%) under Agency Amount Requested.
  4. Click the Recalculate button to calculate and display indirect costs.
  5. Justification must be provided for the following uses: No Indirect Cost, Off Campus Rates, and/or Manual Calculations. Enter this information in the text box.
  6. Select the Save button to save your information.
  7. Select the Reload button to clear all editable fields and return to the previously saved state.

 

KRA Budget: PERMISSIONS

 

Default Permissions

 

Default permissions include: Budget Initiator, Project Director, Co-Project Director, Cost Sharing Orgs, and those included in the approval chain. You may grant additional permissions on an ad-hoc basis by individual or Organization. Sponsored Programs office personnel will have permissions on all Budgets. Please verify that all default individuals with permission to view and modify the Budget and their correct permission settings are listed.

 

Ad-Hoc Permissions

 

Follow these steps to view or modify the current Ad-Hoc Permissions for this Budget:

 

  1. Click the Personnel Lookup magnifying glass to add permissions for an individual.  Select the person from the Universal User Lookup. 
  2. Select Read or Mod access for the person, then click the Add button. 
  3. Click the Chart/Org Lookup magnifying glass to add permissions for an entire Chart/Org.  Select the Chart/Org from the Organization Lookup.
  4. Select Read or Mod access for the Chart/Org, then select the Add button.
  5. Click the Workgroup Lookup magnifying glass to add permissions for a Workgroup.  Select the Workgroup from the Workgroup Lookup.
  6. Select Read or Mod access for the Workgroup, then select the Add button.
  7. Click the Save button to save your information.
  8. Click the Reload button to clear editable fields and return to the previously saved state.
  9. Click the Delete button to remove ad-hoc permissions for a person or Chart/Org.  Default permissions cannot be deleted. 

 

Route Log

 

The Route Log shows listings of Actions Taken, Pending Action Requests, and Future Action Requests for the Budget document. Each action is noted with a time/date stamp, and the name of the person who took the action, or for whom an action is still required. Use the Show buttons to open panels and view details for Pending and Future Action Requests.

 

While the Budget Document does not route as a standalone eDoc, it uses Workflow for permissions functionality. The Budget Document must be designated as Complete on the Budget Overview page before FYI notices are sent to those with permissions. The document will remain in the initiators action list, and cannot be attached to a Kuali Routing form until it is designated as Complete.

 

Workgroup Lookup

 

  1. You must enter at least one character in one of the fields to begin a search. More criteria will narrow the results. Enter any combination of the following search criteria:

a.          Enter the Workgroup Name.

b.          Enter the Workgroup Description.

c.          Enter the ID of a Workgroup Member, or perform a Workgroup Member Lookup.

d.          Enter the Workgroup ID.

e.          Click Active, Inactive, or Show All for the Active Indicator.

  1. Select the Search button after entering the criteria. This will generate the results.
  2. Select a Workgroup from the list of results and click the Return Value link. This will return the selected Workgroup to the Permissions screen.
  3. Select the Clear button to clear the fields of any previously entered criteria and begin a new search.
  4. Click the Return with No Value link to cancel the search and return to the Permissions page.

 

Workgroup Member Lookup

 

  1. You must enter at least one character in one of the fields to begin a search. More criteria will narrow the results. Enter any combination of the following search criteria:

a.          Enter the Last Name.

b.          Enter the First Name.

c.          Enter the Network ID.

d.          Enter the Workflow User ID.

e.          Enter the University ID.

f.            Enter the UUID.

  1. Select the Search button after entering the criteria. This will generate the results.
  2. Select a Workgroup Member from the list of results and click the Return Value link. This will return the selected Workgroup Member ID to the Workgroup Lookup Screen screen.
  3. Select the Clear button to clear the fields of any previously entered criteria and begin a new search.
  4. Click the Return with No Value link to cancel the search and return to the Workgroup Lookup Screen.

 

KRA Budget: OUTPUT

 

Your Budget, including any entered and saved information, will be converted to a Portable Document Format (.pdf) file for output and printing. To be able to read this document, you will need Adobe Acrobat Reader 4.0 or above installed on your computer. Please visit Adobe Acrobat for links and installation instructions. The system displays the output in a new window and provides print and printer options.

  

1. Select the type of output display you wish to view, either: Generic (with High, Medium, or Low level of detail) Agency, or SF 424.

2. Select the Generate PDF Output button. The Budget .pdf document will appear in a new window.

3. From the toolbar along the top of the window, select the print option to print the document.

4. Select the save option to save a copy of the Budget document to your computer or disk.

 

KRA Budget: TEMPLATE

 

The Template screen allows you to make an editable copy of an existing KRA Budget. The system closes the original Budget, displays the new templated Budget in the Budget Parameters page, and assigns it a new tracking number.

 

  1. If you added Ad-Hoc Permissions to the first KRA budget, check the box to copy Ad-Hoc Permissions to the new budget.  If the box remains unchecked, the Ad-Hoc Permissions will not copy over.
  2. Click the checkbox to copy the indirect cost rates from the first KRA budget to the new budget.  If the box remains unchecked and indirect cost rates have changed since the creation of the first budget, the new budget will contain current rates. 
  3. Select the Template button. The new Budget Parameters form will appear displaying copied information.

 

KRA Budget: AUDIT MODE

  

The KRA system offers an auditing function that can be used at anytime during the process of preparing a budget. Audit mode will help you identify required information that is missing or incomplete, and fields that have improperly formatted information (example:  incorrect date format).  Activate the audit mode function to verify information previously entered or to identify information still needed.

Follow these steps: 

 

1. Select the Activate Audit Mode option to turn on this function.

2. Once the Audit Mode function is activated, any errors found will appear on the screen as Error Links.

3. Click on the error link to go back to the form and correct misinformation or to add missing information.

4. Select the Inactivate Audit Mode option to turn off this function.

 

Two types of errors will appear when Audit Mode is activated:  Soft Audit Errors and Hard Audit Errors

 

Soft Audit Mode

 

Soft audit errors are warnings.  When audit mode is activated, soft errors will be listed, but the budget can be submitted into routing (attached to the Routing Form) even if these errors are not corrected.   

 

Hard Audit Mode

 

Hard audit errors must be corrected before the budget can be submitted into routing (attached to the Routing Form).
